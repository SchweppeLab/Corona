Corona requires that your IAPI application is built using [Helios](https://schweppelab.github.io/Helios/) to establish
the connection between application and virtual MS. Helios is a wrapper around IAPI that provides the functionality
of IAPI on multiple instruments, including Corona. Below is a brief overview of adapting your IAPI application to use
Helios and connect to Corona.

#### Step 1: Obtain Helios

The latest release is found [HERE](https://schweppelab.github.io/Helios/download.html).

#### Step 2: Redirect your application to use Helios

Follow the [setup](https://schweppelab.github.io/Helios/setup.html) instructions for Helios. Then update the
application namespaces to use Helios, and use the Helios factory to create an IAPI access container. With rare
exception, all other IAPI code may remain the same.

#### Step 3: Start Corona, then start your IAPI application.

Connection of your IAPI application to Corona is automatic, provided that Corona is already loaded. If an IAPI-supported
instrument is also connected to the computer, your application will connect to the instrument instead of Corona. If using
Corona for IAPI application development, it is recommended to do so on a computer without a paired instrument.
