<h2 class="center">Corona is a virtual mass spectrometer for real-time MS software development.</h2>

<br/>
<img class="img-fluid img-marquee" src="/img/Corona1-ezgif.com-optimize.gif" alt="Corona: A Virtual Mass Spectrometer" />

<br/>

### Features:

- Replay any previously acquired MS run
- Select scan number or retention time subsets
- Operate at normal or accelerated speed
- Queue multiple runs
- Accepts Thermo Raw and mzML data
- Communicates with Thermo IAPI applications using <a class="clink" href="https://schweppelab.github.io/Helios" target="_blank">Helios</a>

<br/>

<h3 class="center narrow">Corona is free and runs on the Windows operating system. <a class="clink" href="https://schweppelab.github.io/Corona/download">Download</a> it now.</h3>
